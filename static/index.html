The above uses a recurrent neural network trained on (a subset of) the Europarl corpus to detect the language in which the source text is written. It uses a rolling 32-byte window over the text, so multiple languages may be detected within a given sample.

I know almost nothing about machine learning, so Keras was invaluable to me in building this thing. I'd like to expand it to cover more languages (perhaps using the Wikipedia dumps or some of These Wikipedia-derived corpora. I expect that significantly expanding the number of detectable languages will require a corresponding significant increase in compute power though.
